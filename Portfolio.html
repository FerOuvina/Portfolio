TO-DO Task Manager

This is a simple, one page TO DO task manager. I made it with vanilla JS, HTML, CSS and a bit of JSON for managing the local storage. The main purpose of this project was to learn how to manipulate and use the local storage api from the browser and to get more practice on manipulating the DOM. In the end it came out to be a real project and it's been adapted to all devices and it's fully functional.

Encryptor

This is a single page text encrypt & decrypt app. I made it using HTML, CSS and vanilla JavaScript, it consist of two main functions, the first one to encrypt the input text from the user, given certain encryption keys, and the second to decrypt the text with the same encryption keys. It also has a copy to clipboard button and it's adapted to mobile devices so it can be accessed from any device.

Hangman Game

This one is a hangman game made with HTML, CSS, and vanilla JavaScript. This was my hardest challenge at that moment because of how much logic it required, it took a lot of research to be done but the result it's great, simple but functional. It also has a built-in keyboard for mobile devices that acts as a real keyboard, I thought of this so I didn't have to put an input on the screen, in this way the user can just type in the screen without the mobile keyboard being on the way.
